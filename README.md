## Sobre

:question: O Rocket.Q é uma aplicação em que os usuários enviam perguntas para salas específicas, onde as questões podem ser respondidas ou excluídas.

:gear: Esse projeto utiliza HTML, CSS, JS, NODEJS e SQL.
